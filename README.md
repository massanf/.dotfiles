### Supported
- linux-gnu
- linux-musl
- darwin (macOS)

### Prerequisites
- sudo
- curl
- bash

### Install
```
curl -s https://raw.githubusercontent.com/massanf/.dotfiles/main/install.sh | sudo bash
```
